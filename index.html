GERENCIAMENTO DE LOJA DE INFORMÁTICA

Este é um projeto de sistema web para gerenciamento de uma loja de informática, desenvolvido em linguagem Java e javaScript, com integração ao banco de dados MySQL. O sistema permite facilitar a gestão da loja, automatizando processos como o cadastro de produtos, controle de estoque, consulta de informações relevantes, atualização de registros e registro de vendas.
